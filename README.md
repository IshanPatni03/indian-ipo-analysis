# Indian IPO Analysis (2020–2025)

## Project Overview
An end-to-end data analysis project examining the listing day and long-term 
performance of 367 Indian mainboard IPOs listed between 2020 and 2025.

## Tools Used
- **Data Collection:** IPOCentral.in, Chittorgarh.com
- **Data Cleaning:** Python (pandas), Excel
- **Database:** MySQL
- **Visualisation:** Power BI

## Key Findings
- Average listing day return across 367 IPOs: **24.1%**
- **73%** of IPOs delivered a positive return on listing day
- **48 IPOs** were listing day heroes (≥30% listing return) but are now 
  averaging **-15.3%** from issue price — all gains wiped out
- **42 IPOs** listed quietly (<30% return) but now average **+290%** 
  from issue price (e.g. Mazagon Dock: +18.6% listing → +3,273% today)
- **2024 is the only year** where average current returns are below 
  listing day returns — suggesting recent IPOs are overpriced
- Only **53.4%** of all IPOs are currently trading above their issue price

## Dataset
- 367 mainboard IPOs (excludes SME, REITs, InvITs, FPOs)
- Sources: IPOCentral.in, Chittorgarh.com Performance Tracker
- Price data as of June 4, 2026
